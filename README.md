# audio_mxing_kiosk

## Instalation

This kiosk is meant to be ready to run upon download from github. Run src/gui.py if no run on startup is available. For new hosts: run "pip install -r requirements.txt. Python 3.12 was used to develop this project.

## Passwords

When you run the kiosk for the first time, you will not be able to access the settings page without a password set for the "admin" user. Running the password_manager.py in a terminal will prompt you to add/change the user password.
